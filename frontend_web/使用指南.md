# AllahPan Web 前端使用指南

## 快速启动

### 1. 启动后端

```powershell
# Windows
cd backend
.venv\Scripts\activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

```bash
# macOS / Linux
cd backend
source ../.venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 打开前端

```powershell
# Windows
cd frontend_web
.venv\Scripts\python run.py
```

```bash
# macOS / Linux
cd frontend_web
source ../.venv/bin/activate
python run.py
```

浏览器自动打开 `http://localhost:3000`，注册账号后即可使用。

## 使用流程

### 1. 注册账号

点击"立即注册"，填写用户名、密码、邮箱，提交后自动进入登录状态。

### 2. 登录

已注册账号可直接登录。勾选"记住登录"可避免刷新后重新登录。

### 3. 上传文件

- **点击上传**：点击上传区域，选择文件
- **拖拽上传**：直接将文件拖入页面任意位置

上传时底部显示实时进度条。

### 4. 下载文件

在文件列表中点击对应文件的"下载"按钮。

### 5. AI 语义搜索

在搜索框输入自然语言描述（如"去年的旅行照片"），500ms 防抖后自动执行搜索。

## API 测试（curl）

```bash
# 注册
curl -X POST http://localhost:8000/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123","email":"admin@example.com"}'

# 登录
curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'

# 文件列表
curl -X GET http://localhost:8000/api/v1/files/list \
  -H "Authorization: Bearer YOUR_TOKEN"

# 上传文件
curl -X POST http://localhost:8000/api/v1/files/upload \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@/path/to/file"

# AI 搜索
curl -X POST http://localhost:8000/api/v1/ai/search \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"query":"风景照片","limit":10}'
```

## 外网远程访问（Cloudflare Tunnel）

### 前提

家庭主机上已安装 `cloudflared`：
- Windows：`winget install cloudflared`
- macOS：`brew install cloudflared`

### 步骤

1. 在 [Cloudflare Zero Trust 控制台](https://one.dash.cloudflare.com/) 创建 Cloudflare Tunnel（选择 Cloudflared），设置公共子域名，URL 指向 `localhost:8000`，复制 Tunnel Token。
2. 在 AllahPan 桌面端 → 设置 → 远程访问，填入 Token，保存并启动。
3. 隧道启动后，状态栏显示公网地址（如 `https://xxx.trycloudflare.com`）。
4. 在任意设备浏览器访问该公网地址即可使用 Web 前端。

### 注意事项

- Tunnel Token 不要泄露或提交到代码库
- 外网访问依赖 JWT 登录，请使用强密码
- 仅在可信任设备上使用"记住登录"
